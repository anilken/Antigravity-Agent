# SEO Strategy

## Keyword Strategy
- **Density:** 0.5-1.5% for primary keywords. Avoid stuffing.
- **Entities:** Map related entities and concepts to build topical authority.
- **LSI:** Use 20-30 semantic variations per topic.

## Metadata Optimization
### Titles
- **Length:** 50-60 Characters.
- **Format:** `Primary Keyword - Hook | Brand`.
- **Content:** Include emotional triggers and numbers for freshness (e.g., "2024").

### Descriptions
- **Length:** 150-160 Characters.
- **Content:** Action-oriented, include primary/secondary keywords, clear CTA.

### URL Structure
- Keep under 60 chars.
- Hyphens, lowercase only.
- `/primary-keyword-topic`

## Structured Data & Snippets
### Featured Snippets
- **Paragraph:** 40-60 words, direct answer, question-based header.
- **List:** 5-8 items, clear header.
- **Table:** Comparison data.

### Schema Markup
- **Priority:** Article, FAQPage, HowTo, Organization, BreadcrumbList.
- **Siloing:** Create topical clusters with parent/child relationships.

## Content Structure
- **H1:** One per page (Main Topic).
- **H2:** Major sections (Secondary Keywords).
- **H3:** Subsections (LSI/Entities).
